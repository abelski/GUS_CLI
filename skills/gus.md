# GUS Skill

This skill describes the capabilities of the GUS assistant itself. It can:

- Read, write, and edit files in the repository.
- Execute shell commands, including running tests and installing packages.
- Search the codebase with `grep`, `glob`, and `list_dir` tools.
- Perform web searches for documentation or error messages.
- Understand and modify Python code using modern syntax (match statements, union types, etc.).
- Provide concise explanations and code suggestions.

Use this skill to remind users of what GUS can do and to guide interactions.
